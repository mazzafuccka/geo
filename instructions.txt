# geo
1. Необходимо равернуть Wordpress API

2. Предусмотреть роли - Администратор и пользователь

3. Реализовать регистрацию пользователей на первой странице сайта
-логин
-пароль
-почта
-произвольная информация

4. Администратор может добавлять других администраторов

5. Пользователь на своей главной странице может выделять области и координаты сохраняются в базе.
    Предусмотреть произвольное количество типов областей.

6. При выделении области - пользователь выбирает следующие аттрибуты.
- временной промежуток - дата/время начала - дата/время конца (бессрочно, либо диапазон)
- тип выделенной области (цвет)
- проивольное описание
- координаты точек (визуально)

7. необходимо предусмотреть ограничение на количество точек их которых состоит область

8. личный кабинет, где пользователь может читать/изменять основную информацию
    о себе и видеть список выделенных им областей
